APB TO PWM macro/IP

Description

Created for LAB2, instantiates an APB-PWM block with 2 16-bit PWM outputs

> Interface:

Address 0x0000 = PWM0 duty cycle 0x0000 - 0xFFFF - read/Write
Address 0x0004 = PWM0 enable - bit0
Address 0x0008 = PWM1 duty cycle 0x0000 - 0xFFFF - read/Write
Address 0x000c = PWM1 enable - bit0

Tested on MAS board

Devices Supported

Certus Pro NX NX / Certus NX / Crosslink NX
