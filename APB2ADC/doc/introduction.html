APB TO ADC macro/IP

Description

Created for LAB12 : Continous ADC, example taken from CertusProNX Versa Board
ADCBLOCK.v created with Radiant IP library creator : ADC_core, external Vref 1.2V, Divider:2

Contrain1 : ADC clock should be equal Fab Clock : example 50Mhz for 1Mbsamp/s
Contrain2 : ADC clock must be from osc4, AND PLL on Lower-Right Mapped
Contrain3 : Top level design requires Analog IO names to be ADC_DP0 ADC_DN0 , ADC_DP1 / ADC_DN1 for right analog port mapping

APB Memory Interface:

Address 0x0000 = ADC0 Result [11:0] (DP0) - read only
Address 0x0004 = ADC0 Select[3:0]- read/write
Address 0x0008 = ADC1 Result [11:0] (DP1) - read only
Address 0x000c = ADC1 Select[3:0]- read/writey

Select[3:0] : 0xB = cP0 / CP1 for both channels. : 0xA = DTR for channel1

Devices Supported

CERTUS PRO NX / CERTUS NX / CROSSLINK NX
