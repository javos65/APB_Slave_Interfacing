APB TO ADC macro/IP

Description

Created for LAB12 : Continous ADC, 10-50Mhz ADC speed, 50MHz state machine
ADV Div=2, 1Msamp/s @ 50Mhz, 200Ksamp/s @ 10Mhz. ADCBLOCK.v created with Radiant IP library creator : ADC_core, Vref 1.2V, Divider:2
ADC clock : 10-0Mhz, from osc4, PLL on Lower-Right PLL Mapped !!

Interface:

Address 0x0000 = ADC0 Result [11:0] (DP0/DN0) - read only
Address 0x0004 = ADC0 Select[3:0]- read/write
Address 0x0008 = ADC1 Result [11:0] (DP1/DN1) - read only
Address 0x000c = ADC1 Select[3:0]- read/writey

Select[3:0] : 0xB = CP0/CN0 or CP1/CN1 for both channels. : 0xA = DTR for channel1

Devices Supported

CERTUS PRO NX
